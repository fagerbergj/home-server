# Client Setup

How to get your Minecraft client ready to join the server.

## Requirements

- Minecraft Java Edition
- [CurseForge launcher](https://www.curseforge.com/download/app) (Windows .exe installer)

## Steps

1. Install the CurseForge launcher and open it
2. Select **Minecraft** from the game list
3. Click **Create Custom Profile**
4. Set the following:
   - **Minecraft version:** `1.21.1`
   - **Mod loader:** `NeoForge`
5. Open the profile and click **Add More Content**
6. Search for and add each of these mods:
   - **Ars Nouveau**
   - **GeckoLib**
   - **Patchouli**
7. Launch Minecraft from the profile
8. Add the server under **Multiplayer → Add Server**:
   - Address: `yourname.asuscomm.com:25565`

## Notes

- You must launch from this CurseForge profile, not the vanilla launcher
- If you join without the mods you'll be kicked immediately
- If the server adds new mods, you'll need to add them to your profile too
